# Instruções de Uso

1. Configure as credenciais da EvolutionAPI nas configurações acima  
   Acesse via http://localhost:8080/manager
2. Adicione ou importe convidados na aba "Convidados"
3. Personalize a mensagem na aba "Mensagem"
4. Exporte a lista de convidados para Excel
5. Agende o envio de convites na aba Convidados, ou clique no botão Enviar Agora
